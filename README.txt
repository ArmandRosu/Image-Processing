Programul incepe citind comanda pe care o primeste.
Acesta citeste intai primul cuvant care este codul comenzii.
Apoi, citeste un numar diferit de cuvinte in functie de comanda introdusa.
Pentru task 1, se pot executa comenzile edit, save si quit.
Edit primeste numele (ex.: poza.bmp) si incarca poza in memorie.
Save primeste locatia unde va fi salvata si cu ce nume...
in cazul nostru fiind folosit folderul output...
(ex.: ./output/alt_nume.bmp).
Quit duce in primul rand la o dezalocare a memoriei ce a fost...
incarcate spre editarea pozei si, mai apoi la inchiderea...
definitiva a programului.
La task 2, este implmentata functia insert, care primeste...
locatia pozei si rezolutia sa, ducand la imserarea acesteia.
Task 3 necesita functiile set si draw.
Cuvantul set va fi urmat fie de termenul draw_color si, mai apoi,...
de codul RGB (red-green-blue) al culorii folosite, fie de line_width...
si de o valoare x, ce reprezita latura patratului ce inconjoara pixelul...
(daca acesta este, bineinteles, diferit de 1 - valoarea default a sa).
In cazul lui draw, exista cazul line, in care se deseneaza o linie...
intre cele doua puncte a caror coordonate le primeste, cazul rectangle...
in care deseneaza un dreptunghi, primind coordonata coltului,...
latimea si inaltimea, respectiv cazul triangle, unde...
primeste coordonatele colturilor triunghiului.
Pe langa acestea exista si functiile predefinite, care primesc...
informatiile fotografiei.
